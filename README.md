# Conway's Game Of Life

Un programme Python qui simule le Jeu de la Vie créé par John Conway

## Remarque

Au départ, j'avais utilisé Tkinter pour la simulation. Mais le module était plutôt destiné à la création d'applications plutôt que la création de jeux. C'est la remarque je me suis fait quand je me suis rendu compte que le programme adapté au module Tkinter est très lent.

## Bonus

J'ai créé une variante du Jeu de la Vie sur Tkinter quand j'étais en Première. Le programme simule un Feu de Forêt.
